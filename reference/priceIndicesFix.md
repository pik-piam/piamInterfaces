# Fixes price indices with wrong reference year

Fixes price indices with wrong reference year

## Usage

``` r
priceIndicesFix(mifdata, priceIndices, referenceYear = 2020)
```

## Arguments

- mifdata:

  file or data that can be converted into quitte object

- priceIndices:

  vector of missing price index variable names

- referenceYear:

  in which index = 1
