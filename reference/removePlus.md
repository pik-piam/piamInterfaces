# Remove \|+\|, \|++\| etc. from variable names

Remove \|+\|, \|++\| etc. from variable names

## Usage

``` r
removePlus(x)
```

## Arguments

- x:

  vector with variable names

## Value

variable names without any plus notation

## Author

Oliver Richters

## Examples

``` r
#' removePlus(c("FE|+|CDR", "FE|CDR|DACCS"))
```
