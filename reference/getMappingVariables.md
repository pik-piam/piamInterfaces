# Retrieves all variables allocated to source potentially used in mappings to project variables

Retrieves all variables allocated to source potentially used in mappings
to project variables

## Usage

``` r
getMappingVariables(project = TRUE, sources = TRUE, addunit = TRUE)
```

## Arguments

- project:

  name of the project of requested mappings such as c('AR6', 'AR6_NGFS')
  or 'mapping.csv'. Use TRUE for all mappings.

- sources:

  model abbreviation(s) used in 'source' column. R = REMIND, M = MAgPIE,
  T = EDGE-T, B = Brick, C = Climate/MAGICC, TRUE = all

- addunit:

  boolean whether units should be appended as 'Variable (Unit)'
